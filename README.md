## How to use
- 5分間で暗算する練習
- 「Start」を押下するとタイマーがスタートし、4桁の足し算が表示される
- 解答を表示されたキーで入力する
- 5分経過すると正答数と不正解数が表示され無効状態になる
- もし、リセット、及びリスタートしたい場合は「リセット」を押下
